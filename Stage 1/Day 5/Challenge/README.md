## Buat BASH Script untuk Instalasi Node Version Manager!
```
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
$ exec bash
$ nvm install 14
$ nvm use 14
$ node -v
$ nvm -v
```
